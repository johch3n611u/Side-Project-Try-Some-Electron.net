# 簡易 xlsx 篩選閱讀器

## 需求

以 xlsx 為簡易資料庫讀取資料做篩選顯示，必須好安裝簡單使用為基準。

預想後續需求不以 sheetjs 純做前端處理，而是搭配 electron.net(.net core & electron) 做後續後端需求的擴增。
